# Compiletime

A collection of helpful compiletime tools for Elixir.

## Installation

The package can be installed as:

  1. Add `compiletime` to your list of dependencies in `mix.exs`:

    ```elixir
    # stable
    def deps do
      [{:compiletime, "~> 0.2.0"}]
    end

    # development
    def deps do
      [{:compiletime, github: "amorphid/compiletime", branch: "0.3.0-dev"}]
    end
    ```
